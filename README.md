## Projetos

### [Sistema de Recomendação de Produtos](./recomendacao-produtos)

Sistema de recomendação baseado em similaridade entre clientes (índice de Jaccard), implementado em **C++** e exposto ao **Python** via pybind11. Inclui duas versões do cálculo de similaridade com comparação de desempenho.

*Tecnologias: C++17, pybind11, Python 3*

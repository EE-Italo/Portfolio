Projetos
Sistema de Recomendação de Produtos

Sistema de recomendação baseado em similaridade entre clientes (índice de Jaccard), implementado em C++ e exposto ao Python via pybind11. Inclui duas versões do cálculo de similaridade com comparação de desempenho.

Tecnologias: C++17, pybind11, Python 3

Análise de Sucesso em Processos Seletivos — Estudantes do CI-UFPB

Análise da taxa de aprovação de estudantes do Centro de Informática da UFPB em processos seletivos de tecnologia, com recorte de gênero e por curso. Os dados foram coletados de fontes oficiais (portais institucionais e Diário Oficial) e enriquecidos com a API de nomes do IBGE para predição de gênero. Projeto em equipe da disciplina de Introdução à Ciência de Dados.

Tecnologias: Python, pandas, numpy, matplotlib, scikit-learn
